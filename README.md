# Scrapergymimg

Este repositorio contiene un script para descargar los PDFs de workouts de [darebee.com](https://darebee.com).

## Uso

Ejecuta el script `scrape_darebee.py` con Python 3:

```bash
python3 scrape_darebee.py
```

El script recorrerá todas las páginas de workouts (de 15 en 15) y descargará cada PDF encontrado en la carpeta `pdfs/`.

> **Nota:** El script requiere acceso a Internet para poder descargar las páginas y los PDFs.
